# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, visible or invisible disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, caste, color, religion, or sexual identity and orientation.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but accept corrections gracefully.
- Keep discussions focused on improving the curation quality.
- Prefer evidence, official sources, and clear caveats over hype.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Personal attacks or trolling.
- Repeatedly submitting low-effort or off-topic resources after maintainers explain the scope.
- Publishing private information without explicit permission.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or restrict participation when behavior harms the project.

For now, report conduct concerns by opening a private channel with the repository owner. If this project gains external maintainers, this file should be updated with a dedicated contact address.
